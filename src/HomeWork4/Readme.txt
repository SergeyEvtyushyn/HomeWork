1. Создан класс DataContainer у которого есть один дженерик <E>. 
2. Внутри DataContainer создано поле Е[] data, предназначеное для хранения передаваемых объектов. 
3. Создан конструктор DataContainer(Е[]). 
4. Создан метод int addData(E item) который добавляет данные во внутреннее поле data и возвращает номер позиции в которую были вставлены данные, начиная с 0.
5. Создан метод Е get(int i). Метод возвращает элемент на искомой позиции.
6. Создан метод Е[] getData(). Данный метод возвращает поле data.
7. Создан метод boolean delete(int i), который удаляет элемент из нашего поля data по индексу. 
8. Создан метод boolean delete(Е item), который удаляет элемент из нашего поля data и возвращает true или false по результатам действия. 
9. Создан метод void sort(Comparator<Е> cmp). Метод сортирует поле по переданному типу Компоратора по возрастанию.
10. Переопределен метод toString(), который выводит содержимое data без ячеек в которых хранится null.
